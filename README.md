# 📊 CRM Data Analysis Dashboard | Power BI Project

## 📌 Overview
This project presents a CRM (Customer Relationship Management) Data Analysis Dashboard developed using Power BI.

The dashboard provides valuable insights into customer behavior, sales performance, revenue trends, customer segmentation, and business growth metrics to support data-driven decision-making.

The project includes:
- Cleaned and transformed CRM dataset
- Interactive Power BI dashboard
- KPI tracking and visual analytics
- Data modeling and DAX calculations

## 🔍 Key Features

### 📈 Key Performance Indicators (KPIs)
- Total Customers
- Total Revenue / Sales
- Average Order Value
- Customer Growth Metrics
- Retention Indicators

### 📊 Visual Analysis
- Revenue trends over time
- Customer distribution by region
- Segment-wise performance analysis
- Product / service comparison
- Sales contribution analysis

### 🎛️ Interactivity
- Slicers for:
  - Time Period
  - Region
  - Customer Segment
  - Product / Category
- Cross-filtering visuals for deep analysis

## 🧹 Data Preparation

Data transformation and cleaning were performed using Power Query:
- Removed duplicates and null values
- Corrected data types
- Created calculated columns
- Built DAX measures and KPIs
- Established data model relationships
